# **App Name**: PortfolioPro

## Core Features:

- Animated Navigation: Animated name, portfolio text, and smooth hover links for navigation.
- Hero Section: Fullscreen landing with animated text intro, tagline, background animation, and 3D glow hover CTAs.
- About Section Animation: Rotating profile image with scroll-triggered animation and self-intro text.
- Project Showcase: Animated project cards with tilt, depth shadow, and slide-up details overlay.
- Skills Animation: Animated skill badges with staggered entry animations and optional radial meters.
- Contact Form Submission: Store contact form submissions to the Firebase Firestore.
- Icon Enhancement: Add generated enhancements (glow/float) to professional/social media icons to emphasize visual appeal.

## Style Guidelines:

- Primary color: Luminous vibrant blue (#43ddff) for a futuristic and glowing aesthetic.
- Background color: Very dark gray (#121417) for a minimalist and luxurious feel.
- Accent color: Soft violet (#b281ff) to highlight interactive elements and CTAs.
- Body and headline font: 'Space Grotesk', a sans-serif with a computerized feel; good for headlines and short amounts of text.
- Glowing icons for contact information, enhancing visual appeal.
- Premium layout with a focus on smooth transitions and 3D depth.
- Every section uses elastic ease, 3D depth, and layered motion for transitions.
